## Installing
Clone this repository into your local machine using the terminal (Mac), CMD (Windows), or a GUI tool like SourceTree.

**Install Dependencies**

```npm install```

**Pre-Deploy**

```npm run build```

**Activate Webpack Dev Server**

```npm run start:dev```
